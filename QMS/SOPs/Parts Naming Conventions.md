Power supply ICs and other regulators , prefix manufacturer part number with PM.


## SMD RESISTORS AND CAPACITORS ##
* capacitor - Example CC1206106050AAAX7R
CC - Prefix
1206 - Package Size
106 - Standard 3 digit value code
050 - Voltage
AAA - Three letter for tolerance, AAA = 10%, AAB = 20%
X7R - Dielectric


* resistor - Example RC0603682075AAA
RC - Prefix for Resistor, Chip
0603 - Package Size
682 - Standard 3 digit value code , not always using this
075 - Voltage
AAA - Three letters for power rating , AAA = 100mW
